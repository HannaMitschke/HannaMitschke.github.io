Buddy Allocator
Introduction to Computer SystemsSolo ProjectC++C


Using starter code, I created a memory allocator in C++ that used the buddy system memory allocation scheme to allocate/free physical memory. In this scheme, the size of the memory blocks are in powers of two and when memory is requested, the blocks are split in half until the smallest block that can still fit the request is filled. When memory is allocated and blocks are split, buddies are created, and when memory is freed, buddies get merged.
